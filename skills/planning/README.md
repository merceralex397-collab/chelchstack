# Planning Skills

This directory contains skills related to project planning, software architecture, task breakdown, and strategic thinking.

## Purpose

Skills in this domain help with:
- Breaking down complex projects into manageable tasks
- Creating implementation plans and roadmaps
- Designing system architecture
- Estimating effort and timelines
- Making technical decisions

## Adding New Skills

Create a subfolder for each skill with the following structure:

```
planning/
└── your-skill-name/
    ├── SKILL.md          # Main skill definition
    ├── agents/           # Agent-specific implementations
    ├── references/       # Documentation, examples
    └── scripts/          # Automation tools
```

## Reference

See the project `AGENTS.md` for skill structure conventions.
