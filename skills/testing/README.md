# Testing Skills

This directory contains skills related to testing strategies, test-driven development, debugging, and quality assurance.

## Purpose

Skills in this domain help with:
- Writing effective tests (unit, integration, e2e)
- Test-driven development workflows
- Debugging strategies and techniques
- Test coverage analysis
- Quality assurance processes

## Adding New Skills

Create a subfolder for each skill with the following structure:

```
testing/
└── your-skill-name/
    ├── SKILL.md          # Main skill definition
    ├── agents/           # Agent-specific implementations
    ├── references/       # Documentation, examples
    └── scripts/          # Automation tools
```

## Reference

See the project `AGENTS.md` for skill structure conventions.
