# UI/UX Skills

This directory contains skills related to UI/UX design, component building, accessibility, and frontend implementation.

## Purpose

Skills in this domain help with:
- Designing user interfaces and experiences
- Building reusable components
- Ensuring accessibility compliance (a11y)
- Responsive design patterns
- Design system implementation
- Animation and interaction design

## Adding New Skills

Create a subfolder for each skill with the following structure:

```
ui/
└── your-skill-name/
    ├── SKILL.md          # Main skill definition
    ├── agents/           # Agent-specific implementations
    ├── references/       # Documentation, examples
    └── scripts/          # Automation tools
```

## Reference

See the project `AGENTS.md` for skill structure conventions.
